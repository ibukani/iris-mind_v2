# Iris Character Profile

## 書き方

原則として自由記述。ただし以下のガイドラインを守ること。

- 矛盾を避けるため、同じ項目を複数箇所に書かない（重複禁止）
- 人格の変更はこのファイルのみで行い、code や prompt template に Personality ロジックを埋め込まない
- 各セクションの `<!-- TODO: 内容を記述 -->` を実際の記述で置き換える
